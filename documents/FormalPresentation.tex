  \addcontentsline{toc}{section}{Abstract}
  \begin{abstract}
    The goal of this project is to design a game engine which will provide the framework for fast, simple deployment of games. This project allows me to learn from several different game engine designs in order to create an engine concrete enough to allow programmers to work on game design rather than programming the structure of windows, graphics, sounds, et cetra, but abstract enough to give the programmer freedom to program a wide range of 2D games. The engine will be available as a static library programmed in C++, which gives the user the power to program in any language that can implement such a library.
  \end{abstract}
